Zero Trust Layer Protocol

A secure, identity-first network overlay that enforces cryptographic authentication before any network state is allocated. Invalid packets are rejected in a three-layer pipeline — making volumetric DDoS attacks structurally ineffective. Current work includes an iOS-first relay-side VIP architecture that moves selected VIP TCP termination off the iPhone Network Extension and onto relay servers. Now with a Go SDK, Docker deployment, HMAC-signed identity headers, and security-hardened infrastructure.
